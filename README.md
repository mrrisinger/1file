## Adding a New Project

1. Drop the `.html` file into the `projects/` folder.
2. Add an entry to `projects.json` at the root:

```json
{
  "name": "My New Tool",
  "description": "One sentence about what it does.",
  "category": "utility",
  "tags": ["optional", "keywords"],
  "created": "YYYY-MM-DD",
  "url": "projects/my-new-tool.html"
}
```

3. Commit and push — GitHub Pages updates automatically.

## Categories

| Value | Use for |
|---|---|
| `utility` | Calculators, checkers, tools |
| `game` | Games, trainers, interactive toys |
| `work-tool` | Job-specific tools |
| `fitness` | Health and fitness trackers |
